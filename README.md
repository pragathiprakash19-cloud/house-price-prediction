# 🏠 House Price Prediction using Machine Learning

## 📌 Overview
This project predicts house prices using a **Linear Regression model** based on:
- Area (sq ft)
- Number of Bedrooms
- Age of the house

It also includes an **interactive web application** built using Streamlit where users can input values and get predicted prices.

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

---

## 📊 Dataset
The dataset contains:
- Area
- Bedrooms
- Age
- Price  

Total records: 20

---

## ⚙️ Project Workflow
1. Data loading and exploration  
2. Handling missing values  
3. Feature selection  
4. Model training using Linear Regression  
5. Model evaluation using:
   - R² Score
   - Mean Squared Error  
6. Visualization of Actual vs Predicted prices  
7. Deployment using Streamlit  

---

## 📈 Results
- R² Score: ~0.99  
- Model shows high accuracy  
- Predictions are close to actual values  

---

## 🚀 Run the App

### Step 1: Install dependencies
pip install pandas scikit-learn streamlit matplotlib

### Step 2: Run Streamlit app
python -m streamlit run app.py
